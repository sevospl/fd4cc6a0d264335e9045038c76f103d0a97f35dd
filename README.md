# Discourse Theme

This is my theme for [Discourse](http://www.discourse.org/).

## Overridden basic color values
* primary: `efefef`
* secondary: `202020`
* tertiary: `6a86bb`
* quaternary: `069`
* header background: `202020`
* header primary: `069`
* hilight: `069`
* danger: `993200`
